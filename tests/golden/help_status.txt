Show system status (gateway, model, workspace)

Usage: rustyclaw status [OPTIONS]

Options:
  -c, --config <PATH>       Path to a config.toml file [env: RUSTYCLAW_CONFIG=]
      --json                Output JSON
      --all                 Show all available information
      --settings-dir <DIR>  Settings directory (default: ~/.rustyclaw) [env: RUSTYCLAW_SETTINGS_DIR=]
      --profile <PROFILE>   Isolate state under ~/.rustyclaw-<PROFILE> [env: RUSTYCLAW_PROFILE=]
      --usage               Include usage statistics
      --no-color            Disable coloured terminal output [env: NO_COLOR=]
  -v, --verbose             Verbose output
      --soul <PATH>         Path to SOUL.md [env: RUSTYCLAW_SOUL=]
      --skills <DIR>        Skills directory [env: RUSTYCLAW_SKILLS=]
      --no-secrets          Disable secrets storage
      --gateway <WS_URL>    Gateway WebSocket URL (ws://â€¦) [env: RUSTYCLAW_GATEWAY=]
  -h, --help                Print help
